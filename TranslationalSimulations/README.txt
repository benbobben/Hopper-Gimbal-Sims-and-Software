
TranslationalAllControllersTest.m - a MATLAB File containing all control laws and an integrator to test and compare results of each control law. 

3 Folders - 
- PID-Programs - Testing just the PIDPWM law
- Maximum Principle - Derivation of the Maximum principle using syms to get a function for time to keep thrusters open. 
- Hysteresis - Testing just for the Hysteresis Law
