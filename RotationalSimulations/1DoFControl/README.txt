
init_1DoFSim - 	Contains initialization values for the Simulink simulations. This includes each control law variables.
		Each Simulink file is a different implementation of a control law in Simulink in one angle(1DoF Pitch) 
		The plant is the same as plant.m from the RotationalSimulations Folder. 

World folder - Contains a X3D file which is a 3D world that is used in the VRBlock from the Aerospace blockset for visualizing systems.
